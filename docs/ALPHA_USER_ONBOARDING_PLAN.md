# Alpha User Onboarding Plan

## Audience

Invite only 3-5 small-B testers after Hosted Online Synthetic E2E passes.

Target profiles:

- hairstyle bloggers
- outfit or styling bloggers
- image consultants
- beauty creators
- photography, makeup, and styling studios

## Onboarding Goal

The Alpha goal is to validate whether creators understand and value the workflow:

```text
Service -> Intake -> Case -> Tags -> Rules -> Report -> Feedback -> Candidate Knowledge
```

The Alpha is not for formal client delivery yet.

## Data Rules

Testers should use synthetic data first.

They must not:

- upload real photos
- enter phone numbers
- enter WeChat IDs
- enter ID card numbers
- enter addresses
- enter private client records
- use the Alpha for paid client service

## Suggested Onboarding Sequence

1. Send short invite.
2. Confirm they understand this is an Alpha test.
3. Send onboarding message.
4. Ask them to complete the test task list with synthetic data.
5. Collect feedback using the summary template.
6. Classify feedback into P0, P1, and P2.
7. Decide whether they can continue to a second test.

## Success Signals

- Creator can explain what the workflow does.
- Creator can complete a synthetic service flow.
- Creator understands why feedback matters.
- Creator sees value in standardized intake and Barber Brief.
- Creator does not ask for unrelated module expansion before Hairstyle stabilizes.
