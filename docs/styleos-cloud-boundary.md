# StyleOS Cloud Boundary

StyleOS Cloud is the planned official hosted product.

## Cloud May Include

- hosted accounts
- team workspaces
- storage
- role permissions
- billing
- higher limits
- managed upgrades
- support
- operational monitoring
- data governance

## CE Boundary

CE defines and may later implement the community workflow. It does not claim to provide the official hosted product.

Small-B users may self-host CE, but most are expected to use Cloud when a managed product is available.
