# Feedback Workflow

Feedback closes the loop between delivery and knowledge improvement.

## Feedback Sources

- creator feedback
- fan/client score
- execution feedback
- report clarity feedback
- barber brief usefulness

## Feedback Use

Feedback can help:

- improve the report template
- adjust rule matching
- identify candidate knowledge
- reject weak rules
- flag Pro candidate patterns

## Privacy Boundary

Feedback should be anonymized before it becomes candidate knowledge.
