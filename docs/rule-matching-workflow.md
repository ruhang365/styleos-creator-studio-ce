# Rule Matching Workflow

Rule matching connects selected tags to relevant StyleOS rule cards.

## Sources

- starter rules from styleos-protocol
- creator working rules
- future Pro rules if the user has authorized access in Cloud

## Steps

1. Read selected tags.
2. Filter rules by module and applicable conditions.
3. Show match reasons.
4. Let creator accept, reject, or override.
5. Send accepted rules to report generation.

## Boundary

Starter rule matches are decision support. They are not expert-certified recommendations.
