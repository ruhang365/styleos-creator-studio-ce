# Tag Workbench Workflow

Tag Workbench helps creators convert raw intake into structured StyleOS tags.

## Inputs

- fan goals
- scenario
- hairstyle attributes
- maintenance willingness
- constraints
- creator notes

## Outputs

- face or proportion tags
- hair attribute tags
- goal tags
- scenario tags
- constraint tags
- uncertainty notes

## Manual Review

Tags should be creator-reviewed. Uncertain signals should stay marked as uncertain.
