# Alpha Known Limitations

This Alpha is intentionally limited.

## Product Limitations

- Only the Hairstyle Workflow is supported.
- There is no payment.
- There is no AI API integration.
- There is no image or photo upload.
- There is no mini program.
- There is no native app.
- There is no team workspace.
- There is no formal StyleOS Pro library.
- There is no expert certification.
- Candidate Knowledge still requires human review.
- The Alpha is not suitable for large-scale public use.

## Operational Limitations

- Access is invite-only.
- Magic link accounts must be approved before use.
- Hosted Alpha does not promise production-grade SLA.
- Testers should use synthetic or low-risk data.
- Cleanup is still a manual operational responsibility.

## Data Limitations

The Alpha should not collect sensitive personal data. It should not store photos, phone numbers, WeChat IDs, ID card numbers, addresses, emails, payment records, or private messages.
