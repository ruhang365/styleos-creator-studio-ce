# Protocol Integration

Creator Studio CE integrates with `ruhang365/styleos-protocol`.

## CE Uses

- input schema
- style tag schema
- rule card schema
- report schema
- starter taxonomies
- starter rules
- synthetic examples
- execution cards

## Integration Principle

CE should consume protocol structures without turning starter rules into verified claims.

## Versioning

Future implementation should track protocol version compatibility in service and report records.

## Boundary

Protocol content is open. StyleOS Pro content is not automatically available to CE.
