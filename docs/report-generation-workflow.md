# Report Generation Workflow

Report generation turns intake, tags, and matched rules into a creator-reviewed Lite Report.

## Inputs

- FanCase
- FanIntake
- StyleTags
- RuleMatches
- creator notes

## Output

- summary
- selected tags
- matched rules
- recommendation
- avoid list
- execution card
- next action

## Manual Edit

The creator should review and edit before delivery. CE should not present draft output as final expert advice.
