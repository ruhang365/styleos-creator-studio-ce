# Creator Workflow

## Steps

1. Create service.
2. Share intake link.
3. Receive fan case.
4. Review tags.
5. Match rule cards.
6. Generate report.
7. Edit manually.
8. Deliver report.
9. Collect feedback.
10. Review candidate knowledge.

## Workflow Principle

The creator remains the decision owner. CE should help standardize workflow and reduce repetitive work, not remove human judgment.

## Privacy Boundary

Fan data should stay in the creator workspace unless explicit consent, anonymization, structuring, and review allow reuse.
