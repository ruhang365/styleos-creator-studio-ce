# Known Issues and Limitations

## Product Limits

- Only Hairstyle Workflow is supported.
- No payment.
- No AI API.
- No photo upload.
- No mini program.
- No native app.
- No team workspace.
- No StyleOS Pro library.
- No expert certification.
- No public self-serve creator onboarding.

## Alpha Limits

- Hosted Online Synthetic E2E has passed and cleanup has been verified.
- Broad real small-B invitations should wait until Alpha UI polish and guided internal testing.
- Candidate Knowledge still requires manual review.
- Alpha testers should use synthetic data first.

## Technical Limits

- Some in-app browser environments may restrict tokenized intake navigation.
- Public token flows should be tested by direct URL or API route when embedded navigation is blocked.
- Supabase Mode depends on hosted environment variables and Supabase Auth redirect configuration.

## Data Limits

- No raw photo storage.
- No phone, WeChat, ID card, address, or private identity data.
- Candidate Knowledge is not verified Pro knowledge.

## Operational Limits

- No production SLA.
- No automated monitoring beyond basic health checks and manual review.
- No automated cleanup for arbitrary test data; marker-based cleanup is required.
