# Alpha Onboarding Message

欢迎参与 StyleOS Creator Studio Alpha。

这轮 Alpha 的目标是测试工作流，不是正式服务客户。请先使用 synthetic 数据完成测试。

请不要输入：

- 真实照片
- 手机号
- 微信号
- 身份证号
- 地址
- 真实客户隐私
- 支付信息

请重点体验：

1. 创建一个 Hairstyle 服务。
2. 用 synthetic 粉丝信息提交 intake。
3. 在 creator 后台查看 case。
4. 生成 tags 和 matched rules。
5. 生成 Lite Report 和 Barber Brief。
6. 提交 synthetic feedback。
7. 观察 Candidate Knowledge 是否能表达可复用经验。

测试完成后，请按反馈模板告诉我们：哪里顺、哪里卡、哪里不符合你的真实服务流程。
