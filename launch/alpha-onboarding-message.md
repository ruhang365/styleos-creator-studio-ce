# Alpha Onboarding Message

欢迎参与 StyleOS Creator Studio Alpha。

这轮 Alpha 的目标是测试工作流，不是正式服务客户。请先使用 synthetic 数据完成测试，不要把它用于真实付费服务。

请不要输入：

- 真实照片
- 手机号
- 微信号
- 身份证号
- 地址
- 真实客户隐私
- 支付信息

请重点体验：

1. 创建一个 Hairstyle 服务。
2. 打开服务详情页里的采集表。
3. 用 synthetic 粉丝信息提交 intake。
4. 在 creator 后台查看 case。
5. 生成 tags 和 matched rules。
6. 生成 Lite Report 和 Barber Brief。
7. 标记报告已交付，并打开分享报告。
8. 提交 synthetic feedback。
9. 观察 Candidate Knowledge 是否能表达可复用经验。

测试时请记录：

- 哪一步你一眼就懂？
- 哪一步让你犹豫？
- 哪些文字不像真实服务场景？
- Lite Report 是否方便你改成自己的表达？
- Barber Brief 是否能帮助你和理发师沟通？
- 如果未来用于真实客户，你最担心什么？

测试完成后，请按反馈模板告诉我们：哪里顺、哪里卡、哪里不符合你的真实服务流程。
