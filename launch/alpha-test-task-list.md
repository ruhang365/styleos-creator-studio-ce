# Alpha Test Task List

## Before Testing

- [ ] Confirm you are using synthetic data.
- [ ] Do not upload real photos.
- [ ] Do not enter phone, WeChat, ID card, address, or private client information.

## Tasks

1. Log in.
2. Open dashboard.
3. Create a synthetic Hairstyle service.
4. Open the service detail page.
5. Submit synthetic fan intake.
6. Open the new case.
7. Generate tags.
8. Match and save rules.
9. Generate Lite Report.
10. Generate Barber Brief.
11. Mark report delivered.
12. Open shared report.
13. Submit synthetic feedback.
14. Open Candidate Knowledge Queue.
15. Review whether the candidate knowledge feels useful.

## Feedback Questions

- Which step was confusing?
- Which step felt valuable?
- Did the Barber Brief feel useful?
- Did the report feel editable enough?
- What would stop you from using this with real clients later?
- What would you need before a second test?
