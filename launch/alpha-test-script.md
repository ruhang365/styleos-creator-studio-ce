# Alpha Test Script

Use synthetic data first. Do not upload real photos or sensitive personal information.

## Steps For Alpha Testers

1. Open the Alpha link.
2. Log in with the provided magic link account.
3. Open Dashboard and confirm you understand the 5-step workflow.
4. Create a hairstyle service.
5. Open the service detail page.
6. Open the intake form from the service page.
7. Submit a synthetic fan intake.
8. Open the new case from Cases.
9. Generate tags.
10. Match and save hairstyle rules.
11. Generate the Lite Report.
12. Generate the Barber Brief.
13. Save the report.
14. Mark the report as delivered.
15. Open the shared report.
16. Submit synthetic feedback.
17. Open Candidate Knowledge Queue.
18. Check whether the candidate knowledge captures a reusable learning.
19. Tell us where the workflow was unclear.

## Synthetic Data Reminder

Use fictional or low-risk test content.

Do not enter:

- real photos
- real sensitive personal data
- phone numbers
- WeChat IDs
- ID card numbers
- addresses
- emails
- payment information
- private conversations

## What To Report

- Did you understand the purpose of the tool?
- Which step helped most?
- Which step felt unnecessary?
- Which page or phrase was confusing?
- Did the Lite Report feel editable enough?
- Did the Barber Brief help you imagine real barber communication?
- Did Candidate Knowledge feel like useful reusable learning?
- Would this help you serve fans if the workflow became more polished?
