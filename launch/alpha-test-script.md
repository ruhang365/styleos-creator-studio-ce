# Alpha Test Script

Use synthetic data first. Do not upload real photos or sensitive personal information.

## Steps For Alpha Testers

1. Open the Alpha link.
2. Log in with the provided magic link account.
3. Create a hairstyle service.
4. Create a synthetic fan case through the intake flow.
5. Review the submitted case.
6. Generate tags.
7. Match hairstyle rules.
8. Generate the Lite Report.
9. Generate the Barber Brief.
10. Mark the report as delivered.
11. Open the shared report.
12. Submit synthetic feedback.
13. Tell us where the workflow was unclear.

## Synthetic Data Reminder

Use fictional or low-risk test content.

Do not enter:

- real photos
- real sensitive personal data
- phone numbers
- WeChat IDs
- ID card numbers
- addresses
- emails
- payment information
- private conversations

## What To Report

- Did you understand the purpose of the tool?
- Which step helped most?
- Which step felt unnecessary?
- Which page or phrase was confusing?
- Would this help you serve fans if the workflow became more polished?
