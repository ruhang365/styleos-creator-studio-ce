# MVP Scope

The MVP focuses on the Hairstyle Suitability Workflow.

## MVP Includes

- Creator account placeholder
- Service menu
- Fan intake form
- Case list
- Case detail
- Tag workbench
- Hairstyle rule matching
- Lite report editor
- Barber brief generator
- Feedback form
- Candidate Knowledge Queue

## MVP Excludes

- Payment
- Full multi-tenant billing
- Real expert model
- Full image AI analysis
- Mini program
- Mobile app
- Full Pro library
- Official certification
- Offline salon integration

## MVP Principle

The MVP should prove that a creator can move from fan intake to tags, rule matches, report, execution card, feedback, and candidate knowledge without relying on private identity data as the core asset.
