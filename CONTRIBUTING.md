# Contributing

StyleOS Creator Studio CE welcomes practical, privacy-safe contributions.

## You Can Contribute

- bug fixes
- workflow improvements
- schema alignment
- UI improvements
- synthetic examples
- documentation

## You Cannot Contribute

- real user photos
- personal data
- unauthorized cases
- API keys
- proprietary Pro content

## Contribution Rules

- Keep changes scoped.
- Use synthetic examples.
- Do not add dependencies without explicit project decision.
- Do not add StyleOS Pro content.
- Do not submit real user data in issues, pull requests, examples, or screenshots.
- Preserve the CE / Cloud / Pro boundary.

## Current Repository Stage

This v0.1 stage is documentation-only. Implementation contributions should wait until the Web App Skeleton milestone.
