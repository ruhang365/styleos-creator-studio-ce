# MVP Scope Lock

## Current MVP Scope

The current MVP is limited to the Hairstyle Workflow.

It includes:

- creator login
- service menu
- fan intake
- case management
- tag generation
- rule matching
- Lite Report
- Barber Brief
- feedback
- Candidate Knowledge Queue

## Current MVP Non-scope

The MVP does not include:

- AI automatic recognition
- image upload
- payment
- mini program
- native app
- multiple module expansion
- StyleOS Pro library
- expert certification
- team workspace
- public creator marketplace

## Scope Rule

Do not expand the MVP until Hosted Online Synthetic E2E passes and Alpha feedback confirms which part of the current workflow needs refinement.
