# Pages

Future page map for Creator Studio CE.

| Page | Purpose | Primary user | Key components | MVP priority |
| --- | --- | --- | --- | --- |
| `/login` | creator sign-in placeholder | Creator | sign-in form, privacy note | P1 |
| `/dashboard` | workspace overview | Creator | case summary, tasks, candidate queue summary | P1 |
| `/services` | list creator services | Creator | service cards, create button | P1 |
| `/services/new` | create service | Creator | service form, module selection | P1 |
| `/services/:id` | manage service | Creator | service details, intake link, settings | P1 |
| `/intake/:serviceId` | fan intake | Fan / Client | intake form, privacy notice | P1 |
| `/cases` | case list | Creator | filters, statuses, service links | P1 |
| `/cases/:caseId` | case detail | Creator | intake summary, status, notes | P1 |
| `/cases/:caseId/tags` | tag workbench | Creator | tag selector, uncertainty notes | P1 |
| `/cases/:caseId/rules` | rule matching | Creator | matched rules, reasons, overrides | P1 |
| `/cases/:caseId/report` | report editor | Creator | lite report editor, execution card | P1 |
| `/reports/:shareId` | delivered report | Fan / Client | read-only report, feedback CTA | P2 |
| `/feedback/:reportId` | feedback form | Fan / Client | rating, comments, execution outcome | P1 |
| `/knowledge/candidates` | candidate queue | Maintainer / Creator | review list, status, evidence | P1 |
| `/knowledge/rules` | public or creator rules | Creator / Maintainer | rule browser, filters | P2 |
| `/settings` | workspace settings | Creator | profile, privacy, export notes | P2 |
