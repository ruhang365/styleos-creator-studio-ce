# MVP User Stories

## Creator

1. As a creator, I want to create a hairstyle service so that fans know what I can deliver.
2. As a creator, I want to share an intake link so that fans submit structured information.
3. As a creator, I want to view all fan cases so that I can manage service progress.
4. As a creator, I want to open a case detail page so that I can review intake answers.
5. As a creator, I want to assign hairstyle tags so that the case becomes structured.
6. As a creator, I want to mark uncertain tags so that I do not overstate confidence.
7. As a creator, I want to match rule cards so that I can explain recommendations.
8. As a creator, I want to reject weak rule matches so that reports stay accurate.
9. As a creator, I want to generate a lite report draft so that I save repetitive writing.
10. As a creator, I want to manually edit the report so that my judgment remains visible.
11. As a creator, I want to generate a barber brief so that the fan can communicate with a hairstylist.
12. As a creator, I want to collect feedback so that I know whether the recommendation helped.
13. As a creator, I want to extract candidate knowledge so that repeated patterns can be reviewed.
14. As a creator, I want to keep private notes separate from public rules so that privacy is protected.

## Fan / Client

15. As a fan, I want to understand what information is required so that I can submit only necessary details.
16. As a fan, I want to see a privacy notice so that I understand how my data is used.
17. As a fan, I want to receive a clear lite report so that I can take action.
18. As a fan, I want to submit feedback so that the creator can improve the service.

## Maintainer

19. As a maintainer, I want to review candidate knowledge so that only safe patterns move forward.
20. As a maintainer, I want to see consent and anonymization status so that privacy rules are enforceable.
21. As a maintainer, I want to classify candidates as open rule, Pro candidate, expert review needed, rejected, or archived.
22. As a maintainer, I want to review schema suggestions so that CE stays aligned with the protocol.
