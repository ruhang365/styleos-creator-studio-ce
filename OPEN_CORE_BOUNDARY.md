# Open-Core Boundary

StyleOS Creator Studio CE follows an open-core model.

## Open-Source CE Includes

- basic workflow
- starter rules integration
- local/self-host deployment path
- lite report generator
- barber brief generator
- candidate knowledge queue structure

## Commercial Cloud May Include

- hosted accounts
- team workspace
- storage
- role permissions
- higher limits
- report export polish
- support
- uptime
- managed upgrades

## StyleOS Pro May Include

- expert-reviewed rules
- verified cases
- advanced knowledge base
- expert model library
- high-conversion templates
- certified creator / salon system
- partner execution network

## Boundary Principle

CE should prove the workflow. Cloud should make it usable for small-B users. Pro should create the advanced knowledge moat.

中文说明：
CE 跑通流程，Cloud 让小 B 直接用，Pro 建立高级壁垒。开源仓库不默认开放 StyleOS Pro、高级知识库、真实案例库、专家模型库和认证体系。
